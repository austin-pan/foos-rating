# Foos Rating Prototype

## Dependencies

```sh
python3 -m pip install -r requirements.txt
```

- `pandas`
- `numpy`

## Execution

Requires `games.csv` to be in the base directory with at least the following
columns:
- `yellow_front`
- `yellow_back`
- `yellow_score`
- `black_front`
- `black_back`
- `black_score`

```sh
python3 foos.py
```
